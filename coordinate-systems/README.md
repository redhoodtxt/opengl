# Shaders

A program to display 10 rotating cubes!

## Description

Using OpenGL to display 10 rotating cubes. MVP, local, world, view and clip spaces. 
## Getting Started

### Dependencies

* `CMake`
* `Make`
### Installing

- just clone this repository and you're good!

### Executing program

```
mkdir build && cd build 
cmake ..
make .
cd bin 
./coordinates
```
